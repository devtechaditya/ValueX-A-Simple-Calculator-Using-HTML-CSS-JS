# ValueX-A-Simple-Calculator-Using-HTML-CSS-JS
📟 Calculator Website

A simple and responsive calculator web application built using HTML, CSS, and JavaScript. This project performs basic arithmetic operations with a clean user interface and smooth user interactions.

The goal of this project is to practice core frontend fundamentals, including DOM manipulation, event handling, and responsive layout design—without using any external libraries or frameworks.

✨ Features

➕ Addition, ➖ subtraction, ✖️ multiplication, ➗ division

🧮 Real-time input and output display

📱 Responsive design (works on desktop and mobile)

⚡ Fast and lightweight (Vanilla JavaScript)

🎯 Clean and intuitive UI

🛠️ Tech Stack

HTML5 – Structure

CSS3 – Styling & layout

JavaScript (ES6) – Logic & interactivity
